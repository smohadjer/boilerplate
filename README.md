Demo: http://boilerplate.saeidmohadjer.com/app/

**Features**
 * Auto deployment of content to production server
 * Supports Sass
 * Validates and minifies CSS and JavaScript resources
 * Precompiles Handlebars templates

**Usage**
 * Cone repo and run: npm install
 * Run: gulp serve
 * Run: gulp build 

 **Requirements**
 * PHP/Apache on production/development server
 * Nodejs on development server
