
#A boilerplate for small PHP Websites

Demo: http://boilerplate.saeidmohadjer.com/

**Features**
 * Auto deployment to production server
 * Supports Sass
 * Validates and minifies CSS and JavaScript resources
 * Precompiles Handlebars templates

**Usage**
 * Run npm install
 * Run gulp serve
 * Run gulp build

 **Requirements**
 * PHP/Apache
 * Nodejs
